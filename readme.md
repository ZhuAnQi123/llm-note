# 🧠 LLM 核心能力 45 天衝刺計劃（基礎加強版）

這是根據你的反饋調整後的學習計劃，將原本的 30 天延長至 45 天，增加了 Python 基礎鞏固、複習日以及更多的實踐環節。

## 核心理念：
1. **實踐為主**：每天必須有可以運行的代碼。
2. **國產模型優先**：使用通義千問 (Qwen) 進行學習，節省成本且方便。
3. **基礎紮實**：補齊 Python 和 NumPy 知識點。

---

## 📅 第 1-2 週：回顧與 Python 基礎強化（補課與鞏固）
*   **Day 1-16**：回顧（如果你已經學過，建議花 1-2 天把之前的代碼重新敲一遍，確保能跑通）。
*   **Day 17**：Python 基礎進階與 NumPy 入門（AI 開發必備）。

*   **Day 20**：第一個完整 CLI 小工具：AI 文本處理助手。

*   **Day 23**：Prompt Engineering 高級技巧：Few-shot 與 Chain of Thought。

*   **Day 25**：複習與 Debug 日：解決代碼中的報錯。

*   **Day 30**：複習日：手動搭建一個本地知識庫問答。

## 📅 第 5 週：LangChain 與 LlamaIndex 框架
